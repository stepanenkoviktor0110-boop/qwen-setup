# Настройка Qwen Code — Оглавление

Пошаговая последовательность. Каждый гайд можно использовать отдельно, но рекомендуется идти по порядку.

## Быстрая установка

Открой **Терминал** (нажми `Cmd + Пробел`, напиши **Terminal**, нажми Enter) и вставь эти 3 строки:

```bash
curl -L https://github.com/stepanenkoviktor0110-boop/qwen-setup/archive/refs/heads/dev.tar.gz | tar xz
cd qwen-setup-dev
bash scripts/setup-qwen.sh
```

Скрипт выполнит все шаги автоматически: установит Node.js, Qwen Code, настроит безопасность, задаст 6 вопросов и установит навыки.
