# Настройка Qwen Code — Оглавление

Пошаговая последовательность. Каждый гайд можно использовать отдельно, но рекомендуется идти по порядку.

## Быстрая установка (одна команда)

```bash
git clone https://github.com/stepanenkoviktor0110-boop/qwen-setup.git
cd qwen-setup
bash scripts/setup-qwen.sh
```

Скрипт выполнит все шаги автоматически. Ниже — подробные гайды для тех, кто хочет разобраться или сделать вручную.
