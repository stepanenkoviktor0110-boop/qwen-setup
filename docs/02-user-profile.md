# Профиль пользователя

Qwen Code может адаптироваться под тебя — общаться проще или сложнее, объяснять подробнее или короче. Для этого нужно ответить на 6 вопросов. Ответы сохраняются в файл `~/.qwen/QWEN.md` — агент читает его при каждом запуске.

---

## Автоматически

Если ты запускал `setup-qwen.sh` — профиль уже заполнен. Проверь:

```bash
cat ~/.qwen/QWEN.md
```

Если видишь секцию **User Profile** — всё на месте. Можно переходить к следующему шагу.

---

## Вручную

Открой файл `~/.qwen/QWEN.md` в любом текстовом редакторе:

```bash
nano ~/.qwen/QWEN.md
```

Добавь в конец файла (или создай файл, если его нет):

```markdown
## User Profile

- **Роль:** [кем работаешь / чем занимаешься]
- **Опыт с кодом:** [не писал / немного пробовал / использую регулярно]
- **Цели:** [что хочешь создавать — боты, сайты, приложения, аналитика, автоматизация]
- **Сфера:** [работа, учёба, хобби, фриланс, свой бизнес]
- **Стиль общения:** [кратко / с объяснениями / пошагово как для новичка]
- **Язык:** [русский / English / оба]
```

Замени текст в квадратных скобках на свои ответы. Сохрани файл (`Ctrl+O`, затем `Ctrl+X` в nano).

---

## Примеры заполнения

**Предприниматель:**
```markdown
- **Роль:** Владелец интернет-магазина
- **Опыт с кодом:** Немного пробовал (HTML, WordPress)
- **Цели:** Боты для Telegram, автоматизация заказов
- **Сфера:** Свой бизнес
- **Стиль общения:** Пошагово, как для новичка
- **Язык:** Русский
```

**Студент:**
```markdown
- **Роль:** Студент-аналитик
- **Опыт с кодом:** Использую регулярно (Python, SQL)
- **Цели:** Аналитика данных, дашборды, автоматизация отчётов
- **Сфера:** Учёба + фриланс
- **Стиль общения:** С объяснениями
- **Язык:** Оба
```

---

## Как это работает

Qwen Code автоматически читает `~/.qwen/QWEN.md` при старте каждой сессии. На основе профиля агент:

- Подбирает уровень объяснений (подробно для новичков, кратко для опытных)
- Предлагает инструменты под твои цели (бот-фреймворки, веб-фреймворки и т.д.)
- Общается на выбранном языке
- Учитывает твою сферу при выборе решений

Профиль можно обновить в любой момент — просто отредактируй файл.
