# Quick Learning — извлечение паттернов

Quick Learning — система, которая учится на ошибках и удачных решениях. После каждой рабочей сессии анализирует, что пошло не так (или неожиданно хорошо), и сохраняет паттерн — правило мышления, которое пригодится в будущем.

---

## Как это работает

1. **Проверка сигналов** — были ли в сессии исправления, изменения плана, откаты? Если всё прошло гладко — ничего не записывает.

2. **Анализ** — для каждого сигнала ищет ошибку мышления: что было упущено, какой подход сработал бы лучше.

3. **Запись** — формулирует правило в виде «триады»:
   - **Триггер** — в какой ситуации возникает ошибка
   - **Действие** — что нужно делать в этой ситуации
   - **Цель** — какую ловушку мышления избегаем

4. **Дедупликация** — проверяет, нет ли уже похожего паттерна. Если есть — увеличивает счётчик, а не дублирует.

---

## Пример

Допустим, агент дважды попал в ситуацию: решение казалось очевидным из-за категории задачи, но при проверке оказалось, что конкретный случай отличается от типичного.

Паттерн:
```
Триггер: решение кажется очевидным из категории объекта
Действие: проверить данные конкретного случая перед применением
Цель: избежать "category default bias" (предвзятость по умолчанию)
```

## Где хранятся паттерны

| Файл | Что содержит |
|------|-------------|
| `~/.qwen/skills/quick-learning/references/reasoning-patterns.md` | Полные записи паттернов |
| `~/.qwen/skills/quick-learning/references/triad-index.md` | Индекс триад (одна строка на паттерн) |

---

## Как запустить

**Автоматически** — quick-learning запускается в конце рабочих сессий, если были сигналы.

**Вручную** — скажи агенту:

```
quick learning
```
или
```
быстрый анализ
```

---

## Установка

Если запускал `setup-qwen.sh` — уже установлен.

Вручную:

```bash
mkdir -p ~/.qwen/skills/quick-learning/references
cp skills/quick-learning/SKILL.md ~/.qwen/skills/quick-learning/
cp skills/quick-learning/references/* ~/.qwen/skills/quick-learning/references/
```

---

## Связь с Skill Master

Когда накопится 25+ паттернов — Quick Learning подскажет запустить Skill Master (`/skill-master`). Skill Master встраивает накопленные правила прямо в навыки агента, чтобы он учитывал их автоматически, а не только при ручном анализе.
