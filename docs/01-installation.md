# Установка Qwen Code на macOS

Qwen Code — AI-помощник для программирования прямо в терминале.
Бесплатно, 1000+ запросов в день.

---

## Шаг 1. Установи Node.js

Node.js — среда для запуска программ на JavaScript. Qwen Code работает на ней.

### Вариант A: Homebrew (если уже есть)

```bash
brew install node@22
```

### Вариант B: Установщик с сайта

1. Открой https://nodejs.org/
2. Нажми зелёную кнопку **"Download Node.js (LTS)"**
3. Скачай `.pkg` файл для macOS
4. Открой скачанный файл и следуй инструкции: **Продолжить → Продолжить → Установить** (введи пароль Mac, если попросит)

### Проверка

Открой **Терминал** (нажми `Cmd + Пробел`, напиши **Terminal** и нажми Enter) и введи:

```bash
node --version
```

Если видишь что-то вроде `v22.15.0` — всё ок. Нужна версия **20 или выше**.

---

## Шаг 2. Установи Qwen Code

В том же терминале:

```bash
npm install -g @qwen-code/qwen-code@latest
```

Подожди 10-30 секунд.

Если появится ошибка с правами доступа (`EACCES`):

```bash
sudo npm install -g @qwen-code/qwen-code@latest
```

Введи пароль от Mac (символы не будут отображаться — это нормально).

### Альтернативный способ — Homebrew

```bash
brew install qwen-code
```

---

## Шаг 3. Запусти

```bash
qwen
```

---

## Шаг 4. Войди в аккаунт

При первом запуске откроется браузер со страницей входа.

- Нажми **"Войти через Google"** или **"Войти через GitHub"**
- Разреши доступ
- Вернись в терминал — он подхватит авторизацию автоматически

## Готово!

Теперь можно писать запросы прямо в терминале:

```
напиши функцию сортировки на Python
```

Qwen Code ответит, напишет код и может создать файлы в твоей папке.

## Частые проблемы

### "qwen: command not found"

Закрой терминал и открой заново. Если не помогло:

```bash
npx @qwen-code/qwen-code@latest
```

### Ошибка прав доступа (EACCES) при установке

Вариант 1 — используй `sudo`:

```bash
sudo npm install -g @qwen-code/qwen-code@latest
```

Вариант 2 — измени владельца папки npm (решает проблему навсегда):

```bash
sudo chown -R $(whoami) $(npm config get prefix)/{lib/node_modules,bin,share}
```

После этого повтори шаг 2 без `sudo`.

### Старая версия Node.js

Если `node --version` показывает версию ниже 20 — обнови:

```bash
brew install node@22
```

Или скачай новую версию с https://nodejs.org/
